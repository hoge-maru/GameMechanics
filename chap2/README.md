GameMechanics


## 2.1、2.2、2.3

* 創発
	* 物理、内部経済、戦術的な操作、社会的インタラクション
* 進行型
	* 事前にデザインされたチャレンジが与えられる

* Bejeweled
* ゲームは創発と進行のハイブリッドが多い
* テトリス、バルダーダッシュ、碁、チェス
	* シンプルなパーツで想像以上に奥深いゲーム
* ゲームが多くのルールを用いることなしに多くの状態を作れるときはプレイヤに理解されやすい
* Connect Fourは重力の要素で奥深いゲームになっている
* シヴィライゼーション
	* 創発型、離散的

# 2.4
	
* 進行のメカニクス
	* レベルの計画、プレイヤがゲームを移動する方法
	* 最初に出会うゲーム要素、進行に必要なタスク
	* ゲーム空間を探検してスキルを上げる
* 最近のゲームは要素が多いのでチュートリアルが必要
* ハーフライフ2
	* 一見自由度が高く見えて、実は一本道
* ゼルダの伝説
	* ハブ＆スポーク型レイアウト
	* ブーメランは武器であると同時に進行ロック開放のキー

# 2.5

* 創発型
	* ルールが少ない、プレイ時間が短い傾向、リプレイバリュー、確率空間大
* 進行型
	* ゲームデザイナが制御しやすい
* StarCraft
	* リアルタイムストラテジー
	* 2はすごいらしい。遊んでみないと説明がよくわからない
* 創発と進行を融合させたゲームの作成は困難

# 2.8 

* 演習1
	* 将棋、オセロ、マインスイーパー、ぷよぷよ、テトリス
* 演習2
	* 中盤・終盤になるにつれてゲームを有利に進められる選択肢が狭まる。悪手が増える。
* 演習3
	* 時間経過によって落下速度が速められるなどのトリガーが存在している
	* 将棋も"持ち時間"が決まっているので、時間経過で終盤に進行するという見方もできる
